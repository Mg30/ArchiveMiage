# Usage 


## Methode 1 : exe

- Se rendre dans le repertoire Executable et double cliquer sur MAIN.

### Depuis le terminal :

`cd/Executable/sudo ./main`


## Methode 2 :

- Il faut avoir la version Python 3.5: si tel n'est pas le cas
sudo apt-get install python3

- Puis depuis le terminal :
  `cd/source`
  `python3 Main.py`
